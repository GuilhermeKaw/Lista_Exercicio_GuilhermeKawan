## Sobre o Projeto

Este repositório contém uma lista de exercícios de lógica de programação em C#, desenvolvida durante atividades educacionais do SENAI.

O objetivo é praticar os fundamentos da programação, incluindo:

- Variáveis e operações básicas  
- Estruturas condicionais  
- Estruturas de repetição (loops)  
- Arrays e Listas  

---

## Objetivo do Evento

O evento tem como foco:

- Desenvolver o raciocínio lógico  
- Introduzir conceitos básicos de programação  
- Preparar os alunos para desenvolvimento de software  
- Aplicar lógica em problemas reais  

---

## Estrutura dos Exercícios

Os exercícios foram organizados em níveis:

### Nível 1 – Variáveis e Operações Básicas
- Calculadora de média  
- Conversor de idade  
- Troca de valores  

### Nível 2 – Estruturas Condicionais
- Verificador de maioridade  
- Classificador de número  
- Validação de login  

### Nível 3 – Estruturas de Repetição
- Contagem regressiva  
- Soma de números ímpares  
- Tabuada personalizada  

### Nível 4 – Coleções (Arrays e Listas)
- Inversão de array  
- Busca e contagem em lista

## Como Executar

1. Abra o Visual Studio  
2. Crie um projeto do tipo Console App (.NET)  
3. Cole o código no arquivo Program.cs  
4. Execute com F5
