## 📌 **Sobre o Projeto**

Este repositório reúne uma **lista estruturada de exercícios de Lógica de Programação em C#**, desenvolvida durante atividades educacionais no **SENAI**.

O projeto tem como finalidade fortalecer a **base lógica necessária para o desenvolvimento de software**, trabalhando desde conceitos fundamentais até estruturas um pouco mais avançadas da linguagem.

Mesmo sendo um projeto acadêmico, ele segue uma organização pensada como se fosse um **projeto real de desenvolvimento**.

---

## 🎯 **Objetivos**

- **Desenvolver raciocínio lógico e pensamento computacional**
- **Aplicar conceitos fundamentais da programação**
- Praticar resolução de problemas
- Criar base sólida para desenvolvimento backend e frontend
- Simular organização de projetos reais

---

## 🧠 **Conteúdos Trabalhados**

### 🔹 **Nível 1 – Fundamentos**

- Declaração de variáveis  
- Operações matemáticas  
- Entrada e saída de dados  
- Conversões de tipos  

### 🔹 **Nível 2 – Estruturas Condicionais**

- **if / else**
- Operadores relacionais  
- Operadores lógicos  
- Validações de fluxo  

### 🔹 **Nível 3 – Estruturas de Repetição**

- **for**
- **while**
- **do while**
- Controle de laços  

### 🔹 **Nível 4 – Coleções**

- **Arrays**
- **List**
- Manipulação de dados em memória  
- Percorrendo coleções

## 🛠️ **Tecnologias Utilizadas**

- **C#**
- **.NET (Console Application)**
- **Visual Studio**

---

## ▶️ **Como Executar o Projeto**

1. Abrir o **Visual Studio**
2. Criar um novo projeto do tipo **Console App (.NET)**
3. Copiar os exercícios para o arquivo **Program.cs**
4. Executar utilizando **F5** ou **Ctrl + F5**
